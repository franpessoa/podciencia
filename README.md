## Stack utilizada

**Front-end:**  Svelte + SASS
\
**Back-end:** Node, SvelteKit, Docker, Fly.io


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/franpessoa/podciencia.git
```

Entre no diretório do projeto

```bash
  cd podciencia
```

Instale as dependências

```bash
  npm i
```

Inicie o servidor

```bash
  npm run dev
```
